# Effect Queues

The effect queues API allows you to manage effect queues in Firebot.

---

## List all effect queues  {{ tag: 'GET', label: '/queues' }}

<Row>
  <Col>
    Retrieves a list of all effect queues
  </Col>

  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/queues">
    ```bash {{ title: 'cURL' }}
    curl http://localhost:7472/api/v1/queues
    ```
    </CodeGroup>

    ```json {{ title: 'Response' }}
    [
      {
        "id": "queue1",
        "name": "Queue 1",
        "status": "active"
      },
      {
        "id": "queue2",
        "name": "Queue 2",
        "status": "paused"
      }
    ]
    ```
  </Col>
</Row>

---

## Retrieve an effect queue  {{ tag: 'GET', label: '/queues/:queueId' }}

<Row>
  <Col>
    Fetches a specific effect queue by its `queueId`.

    ### URL Parameters
    <Properties>
      <Property name="queueId" type="string" required>
        The unique identifier of the queue
      </Property>
    </Properties>
  </Col>

  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/queues/queue1">
    ```bash {{ title: 'cURL' }}
    curl http://localhost:7472/api/v1/queues/queue1
    ```
    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "id": "queue1",
      "name": "Queue 1",
      "status": "active"
    }
    ```
  </Col>
</Row>
