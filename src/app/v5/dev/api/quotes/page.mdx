# Quotes

The Quotes API allows you to manage quotes in Firebot

---

## List all quotes  {{ tag: 'GET', label: '/quotes' }}

<Row>
  <Col>
    Retrieves a list of all quotes
  </Col>

  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/quotes">
    ```bash {{ title: 'cURL' }}
    curl http://localhost:7472/api/v1/quotes
    ```
    </CodeGroup>

    ```json {{ title: 'Response' }}
    [
      {
        "id": 1,
        "text": "This is a sample quote.",
        "originator": "John Doe",
        "creator": "Jane Smith",
        "game": "Game Name",
        "createdAt": "2024-12-14T00:00:00Z"
      },
      {
        "id": 2,
        "text": "Another quote here.",
        "originator": "Alice",
        "creator": "Bob",
        "game": "Another Game",
        "createdAt": "2024-12-13T00:00:00Z"
      }
    ]
    ```
  </Col>
</Row>

## Create a quote  {{ tag: 'POST', label: '/quotes' }}

<Row>
  <Col>
    Creates a new quote.

    ### Request Body
    <Properties>
      <Property name="text" type="string" required>
        The text content of the quote
      </Property>
      <Property name="originator" type="string" required>
        The person who originated the quote
      </Property>
      <Property name="creator" type="string" required>
        The creator of the quote
      </Property>
      <Property name="game" type="string">
        The game context
      </Property>
    </Properties>
  </Col>

  <Col sticky>
    <CodeGroup title="Request" tag="POST" label="/quotes">
    ```json {{ title: 'Request Body' }}
    {
      "text": "This is a new quote.",
      "originator": "Alice",
      "creator": "Bob",
      "game": "Sample Game"
    }
    ```
    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "id": 3,
      "text": "This is a new quote.",
      "originator": "Alice",
      "creator": "Bob",
      "game": "Sample Game",
      "createdAt": "2024-12-14T00:00:00Z"
    }
    ```
  </Col>
</Row>

## Update a quote  {{ tag: 'PUT', label: '/quotes/:quoteId' }}

<Row>
  <Col>
    Creates or updates a specific quote. If a quote with the provided `quoteId` does not exist, it will be created;
    otherwise, it will be updated.

    ### Request Parameters
    <Properties>
      <Property name="quoteId" type="string" required>
        The unique identifier of the quote to be updated or created
      </Property>
    </Properties>

    ### Request Body
    <Properties>
      <Property name="text" type="string" required>
        The text content of the quote
      </Property>
      <Property name="originator" type="string" required>
        The person who originated the quote
      </Property>
      <Property name="creator" type="string" required>
        The creator of the quote
      </Property>
      <Property name="game" type="string">
        The game context
      </Property>
      <Property name="createdAt" type="string">
        The creation date
      </Property>
    </Properties>
  </Col>

  <Col sticky>
    <CodeGroup title="Request" tag="PUT" label="/quotes/1">
    ```json {{ title: 'Request Body' }}
    {
      "text": "Updated quote content",
      "originator": "Alice",
      "creator": "Bob",
      "game": "Sample Game",
      "createdAt": "2024-12-14T00:00:00Z"
    }
    ```
    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "id": 1,
      "text": "Updated quote content",
      "originator": "Alice",
      "creator": "Bob",
      "game": "Sample Game",
      "createdAt": "2024-12-14T00:00:00Z"
    }
    ```
  </Col>
</Row>

## Partially update a quote  {{ tag: 'PATCH', label: '/quotes/:quoteId' }}

<Row>
  <Col>
    Partially updates an existing quote. You can update specific fields such as `text`, `originator`, `creator`, `game`,
    or `createdAt`.

    ### Request Parameters
    <Properties>
      <Property name="quoteId" type="string" required>
        The unique identifier of the quote to be updated
      </Property>
    </Properties>

    ### Request Body
    <Properties>
      <Property name="text" type="string">
        The text content of the quote
      </Property>
      <Property name="originator" type="string">
        The person who originated the quote
      </Property>
      <Property name="creator" type="string">
        The creator of the quote
      </Property>
      <Property name="game" type="string">
        The game context
      </Property>
      <Property name="createdAt" type="string">
        The creation date
      </Property>
    </Properties>
  </Col>

  <Col sticky>
    <CodeGroup title="Request" tag="PATCH" label="/quotes/1">
    ```json {{ title: 'Request Body' }}
    {
      "text": "Partially updated quote content",
      "originator": "Alice"
    }
    ```
    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "id": 1,
      "text": "Partially updated quote content",
      "originator": "Alice",
      "creator": "Bob",
      "game": "Sample Game",
      "createdAt": "2024-12-14T00:00:00Z"
    }
    ```
  </Col>
</Row>
