# Timers

The timers API allows you to manage timers in Firebot.

---

## List all timers  {{ tag: 'GET', label: '/timers' }}

<Row>
  <Col>
    Retrieves a list of all timers
  </Col>

  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/timers">
    ```bash {{ title: 'cURL' }}
    curl http://localhost:7472/api/v1/timers
    ```
    </CodeGroup>

    ```json {{ title: 'Response' }}
    [
      {
        "id": "timer1",
        "name": "Timer 1",
        "active": true
      },
      {
        "id": "timer2",
        "name": "Timer 2",
        "active": false
      }
    ]
    ```
  </Col>
</Row>

---

## Retrieve a timer  {{ tag: 'GET', label: '/timers/:timerId' }}

<Row>
  <Col>
    Fetches a specific timer by its `timerId`.

    ### URL Parameters
    <Properties>
      <Property name="timerId" type="string" required>
        The unique identifier of the timer
      </Property>
    </Properties>
  </Col>

  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/timers/timer1">
    ```bash {{ title: 'cURL' }}
    curl http://localhost:7472/api/v1/timers/timer1
    ```
    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "id": "timer1",
      "name": "Timer 1",
      "active": true
    }
    ```
  </Col>
</Row>
